# VolunteerNow

Find volunteer opportunities in your community — no sign-up required.

VolunteerNow lets you search for local organizations where you can give back. Enter any city, neighborhood, or ZIP code to discover nearby places like food banks, community centers, animal shelters, libraries, and more. Browse results by cause category and distance, then connect with organizations directly.

## Features

- **Instant search** — no account or login needed
- **Cause filtering** — filter by environment, education, health, animals, food, or community
- **Distance control** — adjust search radius to find opportunities near or far
- **Dark mode** — toggle light and dark themes
- **Direct links** — open any place on Google Maps or OpenStreetMap

## Tech Stack

- [React](https://react.dev/) 18 + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/) for building
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- [React Router](https://reactrouter.com/) (HashRouter)
- [Supabase](https://supabase.com/) for edge function proxy (Nominatim + Overpass API)
- [TanStack React Query](https://tanstack.com/query)
